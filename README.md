# TreePlant - Social Development Platform

**Live Site URL:** [https://tree-plantation-9ac31.web.app/](https://tree-plantation-9ac31.web.app/)

## Website Features
* **Event Management:** Users can easily create, update, and manage their own social development events like tree plantations and urban cleanups.
* **Community Participation:** Logged-in users can join events organized by others with a single click and track their joined events in a private dashboard.
* **Search & Filter System:** Includes a robust search functionality to find events by name and filter them by specific categories (e.g., Plantation, Relief, Cleanup).
* **Secure Authentication:** Features a secure login system using Email/Password and Google Social Login, ensuring private routes are protected.
* **Good User Experience:** Built with a fully responsive design, smooth animations, and a toggleable Dark/Light theme for a comfortable viewing experience.
